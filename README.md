# CHRONOS - Luxury Timepieces

A premium, modern landing page for a fictional luxury watch brand, CHRONOS. This project features a stunning, interactive UI designed to evoke elegance, precision, and high-end craftsmanship.

## Features

- **Immersive Hero Section**: An animated starfield canvas background with an aurora effect and shooting stars.
- **Dynamic Preloader**: A customized loading screen that enhances the premium feel before the page fully reveals itself.
- **Interactive UI Elements**: Smooth scroll animations, hover effects, and a responsive navigation bar.
- **Interactive Product Showcase**: A curated collection section with product cards, featuring ratings, pricing, and "Quick View" / "Add to Cart" functionality.
- **Engaging Animations**: Floating product imagery, slide-up text animations, and dynamic statistics counters.
- **Modern Responsive Layout**: Built with a mobile-first approach, ensuring the site looks flawless on devices of all sizes.

## Technologies Used

- **HTML5**: Semantic structure and accessible markup.
- **CSS3**: Custom styling, CSS Grid & Flexbox layouts, intricate animations, and responsive design.
- **JavaScript**: Interactive elements like the canvas starfield background, scroll animations, preloader logic, and dynamic counters.

## Getting Started

To view the project locally:

1. Clone or download the repository to your local machine.
2. Open the index.html file in any modern web browser.
3. No build tools or package managers are required!

## Project Structure

- `index.html`: The main structured HTML document containing all page sections.
- `styles.css`: The styling rules, variables, and animations that give the page its premium look.
- `script.js`: The JavaScript logic for dynamic background effects, preloader, and interactive UI components.
